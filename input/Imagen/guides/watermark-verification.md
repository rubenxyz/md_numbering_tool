This page describes how to verify an image watermark in Imagen on Vertex AI.

By default, Imagen on Vertex AI adds a digital watermark any images generated by a [model version](https://cloud.google.com/vertex-ai/generative-ai/docs/models#imagen-models) that supports watermarking generated images. When you generate an image with a watermark, Imagen adds a non-visible digital watermark, called a [SynthID](https://deepmind.google/technologies/synthid), to images.

You can verify if an image contains a digital watermark or not.

## Verify a watermarked image

Do the following:

1.  In the Google Cloud console, go to the **Vertex AI \> Media Studio** page.
    
    [Go to Media Studio](https://console.cloud.google.com/vertex-ai/studio/media/generate;tab=image)
    
2.  Click **Upload**. In the displayed file dialog, select a file to upload.
    
    Watermarked images display a **SynthID detected** badge.
    
    ![Sample verified watermark in generated image in console](https://cloud.google.com/static/vertex-ai/generative-ai/docs/image/images/quickstart-watermark-detected.png)
